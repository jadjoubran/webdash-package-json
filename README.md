# Webdash Package JSON

![Package json preview](https://i.imgur.com/hBikrcr.png)

```bash
npm install --save-dev webdash-package-json
```

## Features

* Visualize your package.json
* See installed versions
* One-click update
* Follows your version constraints
* Works with `npm` & `yarn`
